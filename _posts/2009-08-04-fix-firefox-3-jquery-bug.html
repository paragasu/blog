It is the problem when the $(document).ready() executed before the CSS loaded.
Make sure all CSS come before the scripts


<script type="text/javascript" src="">
<link rel="stylesheet" type="text/css" href="">

to


<link rel="stylesheet" type="text/css" href="">
<script type="text/javascript" src="">

fix ff3 issues
