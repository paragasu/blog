<?php
$func = "sqrt";
if (is_callable($func)) {
print $func(49);
}
//result 7
?>

Alternatively you can use call_user_func() and call_user_func_array()

call_user_func is twice slower than direct call function and call_user_func_array is 10%-20% slower than call_user_func
